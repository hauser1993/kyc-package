ðŸ”§ Configuration

Here's how to configure the KYC flow for your website:

// 1. Include the SDK script
<script crossorigin src="https://cdn.ballerine.io/js/1.5.112/ballerine-sdk.umd.cjs"></script>

// 2. Configure and initialize
const config = {
  endUserInfo: {
    id: 'unique-user-id',
    firstName: 'John',
    lastName: 'Doe',
    email: 'john@example.com'
  },
  backendConfig: {
    baseUrl: 'https://your-backend-url.com',
    auth: {
      method: 'jwt',
      authorizationHeader: 'Bearer your-jwt-token'
    }
  },
  uiConfig: {
    general: {
      colors: {
        primary: '#667eea'
      }
    }
  }
};

// 3. Initialize and mount
await BallerineSDK.flows.init(config);
await BallerineSDK.flows.mount({
  flowName: 'my-kyc-flow',
  elementId: 'kyc-flow',
  callbacks: {
    onFlowComplete: (payload) => console.log('KYC Complete:', payload),
    onFlowError: (payload) => console.log('KYC Error:', payload)
  }
});
